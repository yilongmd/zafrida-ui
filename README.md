<!-- Plugin description -->
A Frida UI tool window inside PyCharm / JetBrains IDEs.

This plugin provides a dedicated tool window for Frida, a dynamic instrumentation toolkit. It allows developers to interact with Frida directly from their JetBrains IDE, making it easier to instrument and analyze applications during development.
<!-- Plugin description end -->

## Installation

- Using IDE built-in plugin system:
  
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "zafrida-ui"</kbd> >
  <kbd>Install Plugin</kbd>
  
- Manually:

  Download the [latest release](https://github.com/yilongmd/zafrida-ui/releases/latest) and install it manually using
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>
